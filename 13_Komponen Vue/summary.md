# 13_Komponen Vue
## Resume
komponen merupakan suatu sistem template untuk memperluas element
komponen vue dapat di gunakan kembali dengan tujuan :
1. Lebih rapi
2. Lebih mudah di rawat
3. Lebih berkualitas
untuk mengirim data dari induk keanak dapat menggunakan "msg"
